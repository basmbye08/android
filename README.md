📌 Overview

This project contains a collection of Android applications developed as part of the Android Application Development Lab. Each task demonstrates different core concepts of Android development including UI design, user input handling, notifications, intents, file handling, and menus.

🚀 Features / Experiments
🔹 Q1: User Input & Display

Takes user input from a text field

Displays a personalized greeting message on button click

🔹 Q2: User Form Application

Collects user details:

Name, Password, Address

Gender (Radio Buttons)

Age & Date of Birth

State (Spinner)

Displays entered data after submission

Implemented using:

Linear Layout

Relative Layout

Table Layout

🔹 Q3: Notification System

Sends notifications to the notification panel

Displays a Toast message with notification details

🔹 Q4: Intent-Based Menu App

Performs actions using Implicit Intents:

Dial a phone number

Open a website

Send an SMS

🔹 Q5: Login System with File Storage

Stores user credentials in a text file

Verifies login details

Displays:

✅ Login Successful

❌ Login Failed

🔹 Q6: Activity Navigation

Demonstrates:

Explicit Intents (Activity switching)

Implicit Intents (Web, Dial, Map)

🔹 Q7: Options Menu

Implements a menu with multiple options:

Home, Search, Settings, Share, About, Exit

Includes dialog boxes and toast messages

🔹 Q8: UI Controls

Uses:

CheckBoxes (Hobbies)

RadioButtons (Gender)

RadioGroups (Age Range)

Displays selected options dynamically

🛠️ Technologies Used

Language: Java

Framework: Android SDK

IDE: Android Studio

UI Components: XML Layouts

📷 Output

Each experiment includes output screens demonstrating the functionality of the application.
